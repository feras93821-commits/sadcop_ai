# الأسئلة الشائعة والمحادثات الطبيعية

**مرحبا / هاي / سلام عليكم**
أهلاً وسهلاً فيك! كيفك؟ كيف أقدر أساعدك اليوم؟

**كيفك؟ / شلونك؟ / كيف الحال؟ / شو أخبارك؟**
الحمدلله تمام، وأنت كيفك؟

**من أنت؟ / شو أنت؟ / مين أنت؟**
أنا مساعد الشركة السورية للبترول - فرع محروقات اللاذقية. جاهز أجاوبك على كل أسئلتك.

**أين بنزين؟ / وين بنزين؟ / بنزين موجود؟**
قلي وين أنت بالضبط (أي منطقة) وأعطيك أقرب محطة فيها بنزين.

**فيه مازوت؟ / وين مازوت؟ / مازوت موجود؟**
قلي أي منطقة أنت فيها وراح أعطيك آخر خبر عن توفر المازوت.

**كم سعر البنزين؟ / شو سعر البنزين؟**
الأسعار تتغير دورياً. قلي أي نوع (90 أو 95 أو ديزل) وأعطيك آخر سعر.

**أين غاز؟ / وين غاز؟ / غاز موجود؟**
فيه غاز متوفر. قلي منطقتك وأعطيك أقرب نقطة توزيع.

**بدي أشتكي / عندي شكوى**
تفضل اكتب شكواك وأنا أسجلها لك فوراً وراح توصل للإدارة.

**شو أخبار التوزيع؟ / فيه توزيع اليوم؟**
رح أشيك آخر منشورات صفحة الفيسبوك وأعطيك آخر خبر.

**أنت بتعرف شي؟ / عندك خبر؟**
أنا عندي كل معلومات المحطات والتوزيعات، اسألني اللي بدك إياه.
